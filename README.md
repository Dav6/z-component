
### 安装插件

```sh
npm install
```

### 运行demo

```sh
npm run dev
```

### 打包插件

```sh
npm run build
```

### 打包提交
```sh
git push 
git commit -m "menu-model 列表增加icon 参数, isCollapse 是否切换伸缩   事假回调 onCollapseClick 接口变化的值 " 
git add . 
npm run build 

```
