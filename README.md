
### 安装插件

```sh
npm install
```

### 运行demo

```sh
npm run dev
```

### 打包插件

```sh
npm run build
```

### 打包提交
```sh
git push 
git commit -m "table-model 表头上的按钮显示 根据 sectionButtons 字段显示，不在显示默认 delete 按钮，但是设置key为delete 会有默认参数 " 
git add . 
npm run build 

```
